# Nuxt and Tailwind Minimal Starter

A repo with Nuxt 3 and Tailwind pre-installed

## Setup

Clone the repositiory
Make sure to install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```
